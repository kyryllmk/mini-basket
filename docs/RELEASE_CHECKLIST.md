# Release Checklist

Use this checklist before pushing or publishing the portfolio repository.

- [ ] `npm install` completes successfully.
- [ ] `npm run typecheck` passes.
- [ ] `npm run build` passes.
- [ ] App runs locally with `npm run dev`.
- [ ] Solo mode checked from start to game over.
- [ ] 1v1 AI mode checked from start to game over.
- [ ] No visible console errors during normal gameplay.
- [ ] Responsive layout checked on desktop width.
- [ ] Responsive layout checked on mobile width.
- [ ] Touch/mobile controls verified or documented as limited.
- [ ] Asset licenses checked.
- [ ] `docs/ASSET_CREDITS.md` completed.
- [ ] README complete.
- [ ] Screenshots added to README.
- [ ] Demo link added to README.
- [ ] GitHub repo description added.
- [ ] GitHub repo topics added.
- [ ] GitHub issue templates present.
- [ ] GitHub Actions CI passes.
